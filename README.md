Air Quality Index Search Engine
A full-stack web application that lets you check real-time air quality for any city around the world. Built with a Node.js backend and clean, modern frontend.

What It Does
Search by City: Type any city name and get instant air quality data

Detailed Air Quality Info: See AQI numbers, pollution levels, and health recommendations

Smart Caching: Faster results for cities you've searched before

Beautiful Design: Clean, responsive interface that works on all devices

Tech Stack
Backend: Node.js + Express.js

Custom caching system with automatic expiry

RESTful API design

Error handling and rate limiting

Frontend: Pure HTML, CSS, JavaScript

No frameworks - fast and lightweight

Responsive design

Real-time search with loading states

API: World Air Quality Index (WAQI) - provides real-time air quality data
